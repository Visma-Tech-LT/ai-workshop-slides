06 · Security

What Never
Goes in Context

Anyone who can inject tokens into context gains control of your agent. Treat context like a public API.

Deny always wins. Even hooks returning "allow" cannot override deny rules.

🔑

API Keys & Tokens

sk-* · ghp_* · AKIA*
Use env vars instead.

🔐

Private Keys & Certs

SSH · TLS · PGP
Never read ~/.ssh/

🗄️

DB Credentials

Connection strings · passwords
Use Vault or env vars.

👤

PII & Health Data

Anonymize before passing
to any agent or model.
