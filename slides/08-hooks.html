04 · Hooks

Deterministic
Guards

Shell scripts or LLM evals that fire at lifecycle events. Unlike skills (suggestions), hooks always execute.

PreToolUse PostToolUse SessionStart Stop PermissionRequest FileChanged

Exit code 2 blocks the operation — skills can't override this.

.claude/settings.json — run tests before every commit
{
  "hooks": {
    "PreToolUse": [{
      "matcher": "Bash(git commit*)",
      "hooks": [{
        "type": "command",
        "command": "bash .claude/hooks/run-tests.sh"
      }]
    }]
  }
}
git commit
→
PreToolUse
→
npm test
→
pass → commit
fail → blocked
.claude/hooks/run-tests.sh
#!/bin/bash
echo "🧪 Running tests before commit..."

npm test -- --watchAll=false --passWithNoTests

if [ $? -ne 0 ]; then
  echo "❌ Tests failed — commit blocked" >&2
  exit 2  # BLOCKS the git commit
fi

echo "✅ All tests passed"
