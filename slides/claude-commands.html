Useful Claude Code Commands

/rewind Undo the last assistant action — rolls back file changes and removes the last exchange from context.
/clear Wipe the conversation context. Start fresh without closing the session.
/compact Summarise the conversation and compress it — keeps context under the limit without losing thread.
/resume Resume a previous Claude Code session by picking from recent conversation history.
/model Switch model mid-session — Opus 4.7 for hard reasoning, Sonnet 4.6 for speed.
/memory View and edit Claude's persistent memory — facts it carries across all sessions.
/advisor Switch to advisor mode — Claude analyses and suggests without touching any files.
/plan Enter plan mode — propose a full implementation plan and get sign-off before writing code.
! <cmd> Run a shell command inline — output lands in the conversation so Claude can act on it.
@ <file> Pull a file or folder directly into context — useful for pinning a spec or plan without copy-paste.
