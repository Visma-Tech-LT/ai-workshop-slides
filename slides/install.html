Setup · Install

Get Claude Code Running

STEP 1 · INSTALL CLI

Follow the quickstart at code.claude.com:

code.claude.com/docs/en/quickstart
STEP 2 · SET API KEY

You'll receive an ANTHROPIC_API_KEY — set it as a global environment variable using the instructions on the right.

STEP 3 · VERIFY

Restart your terminal and run claude — it should open without asking for a key.

🍎 macOS / Linux — add to ~/.zshrc or ~/.bashrc
export ANTHROPIC_API_KEY=<your-key>
🪟 Windows — System Environment Variables
1. Search "env" in the Start menu
   → Edit the system environment variables

2. Click Environment Variables…

3. Under User variables, click New
   Name:  ANTHROPIC_API_KEY
   Value: <your-key>

4. Click OK on all windows
   Restart your terminal
